# Resumen de licencias

> **NOTA:** Desde el 3 de marzo de 2025, tanto PHP Tools para Visual Studio como Visual Studio Code utilizan el mismo modelo de licencia explicado a continuación. Las versiones anteriores no se ven afectadas y mantienen el modelo de licencia vigente en el momento de su lanzamiento. 

# Modelo de Licencia Perpetua con Fallback

Utilizamos un **modelo de licencia perpetua con fallback** para PHP Tools tanto en Visual Studio como en Visual Studio Code. Esto significa:

- Al comprar una **suscripción anual**, obtienes inmediatamente una licencia perpetua fallback para la versión disponible en el momento de la compra.
- Durante tu suscripción activa, tienes acceso a todas las actualizaciones y soporte.
- Si **decides no renovar**, puedes seguir usando la versión disponible **12 meses antes de la fecha de expiración de la suscripción**, aunque no recibirás actualizaciones ni soporte de nuevas versiones.

- Para **suscripciones mensuales**, después de haber pagado 12 meses consecutivos, recibirás una licencia perpetua fallback para la versión disponible cuando comenzó tu suscripción. Tras la expiración, puedes seguir usando esta versión de fallback, pero no tendrás acceso a actualizaciones ni soporte.

## Tipos de licencia

La **licencia comercial** es la opción para entidades legales, incluidas empresas, organizaciones (con o sin fines de lucro) y gobiernos. Puede ser instalada por cualquier desarrollador en cualquier máquina dentro de una empresa u organización, siempre que el número total de desarrolladores que usen el software al mismo tiempo no supere el número de licencias adquiridas. Consulta más información en [Organizaciones - Licencia Comercial](./commercial-license). 

La **licencia personal** es la opción para desarrolladores individuales que la adquieren con sus propios fondos. No es transferible. El producto puede instalarse en diferentes computadoras, pero no puede utilizarse en ellas al mismo tiempo. Una licencia personal no está disponible para empresas ni para individuos que esperen el reembolso del costo de la licencia por parte de una empresa. Consulta más información en [Individuos - Licencia Personal](./personal-license).
