# 商用ライセンス

## 商用ライセンスは何か。

**商用ライセンス**は、企業または組織のオプションです。同時にソフトウェアを使用する開発者の総数が購入したライセンス数を超えない限り、任意の開発には企業または組織内の任意の機器にインストールし使用することができます。

## 商用ライセンスを購入すると何が得られますか？

商用ライセンスでは、以下の3つのオプションから選択できます：  

1.) PHP Tools for Visual Studio  
2.) PHP Tools for VS Code  
3.) PHP Tools for All Platforms (VS + VS Code)  

すべてのオプションには以下が含まれます：  

- **12か月のサブスクリプション**（購入したプラットフォームに対し、すべてのアップデートおよび新バージョンへのアクセスを含む）  
- **12か月のプロフェッショナルサポート***  
- **永久バックアップライセンス**（購入時またはサブスクリプション終了の12か月前のバージョンを無期限で使用可能）  

**新機能:** 月額サブスクリプションも利用可能です。  

- 月額課金で、サブスクリプションが有効な間は継続的にアップデートとサポートにアクセス可能  
- **永久バックアップライセンス**は、**12か月連続でサブスクリプションを維持した場合**に自動で付与  
- 12か月未満で月額サブスクリプションを中止した場合、永久バックアップライセンスは付与されません  

12か月の期間（年間または月額）が終了した後、更新を行わない場合は、アップデートやサポートを受け取ることはできませんが、永久バックアップライセンスでカバーされているバージョンは引き続き使用できます。  

価格や購入オプションの詳細については、[こちら](https://www.devsense.com/purchase)をクリックしてください。  

*サポートは、Visual Studio 2010-2022またはVS Codeで動作するDevsenseのツールに関するものであり、Visual StudioのIDE自体のサポートではありません。  

## 商用ライセンスは異なる機器で使用できますか。

はい、同じ企業や組織内で異なる機器で使用することができます。ただし、アクティブなユーザーの数はライセンスの数を超えることはできません。複数のユーザーが同時に同じライセンスを使用することはできません。つまり、5つのライセンスがある場合、アクティブユーザーは5人を超えることはできません。各アクティブユーザーには、独自のライセンスが必要です。

## 複数の従業員が同じ商用ライセンスを使用できますか。

はい。2人のユーザーが同時に使用しない限り、1つのライセンスを共有で使用できます。各アクティブユーザーには、独自のライセンスが必要です。

## 私の企業に従業員が12名います。ライセンスをいくつ購入すべきですか。 

それはすべて、ソフトウェアのアクティブユーザーによります。複数のユーザーが同時に同じライセンスを使用することはできません。すべての開発者が同時に作業している場合、12のライセンスが必要です。ただし、2人のユーザーが同時に使用しない限り、1つのライセンスを共有で使用できます。各アクティブユーザーには、独自のライセンスが必要です。

## 自宅で商用ライセンスを使用できますか。

それはお客様の会社ポリシーによります。必ず最初にマネージャーに確認してください。
