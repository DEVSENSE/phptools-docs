# 購入および支払い

## 個人開発者向けにPHPツールを購入するプロセスはどのようになっていますか。 

注文プロセスが簡単にできます：

1.	[購入ページ](https://www.devsense.com/purchase) に移動し、「個人」タブを選択します。
2.	購入するライセンスの種類を選択します：ビジュアルスタジオ用、ビジュアルスタジオコード、又は両方。選択したライセンスの下にある「今すぐ購入」ボタンをクリックします。
3.	これで、ショッピングカートに商品が表示されます。選択したものが正しい場合、下にスクロールし、請求情報と支払いオプションの入力を続けます。
4.	支払いオプション（デビット/クレジットカード、銀行振込など）を選択し、「続く」をクリックして注文を完了します。
5.	注文が行われ、支払いが承認されると、ライセンスキー番号が電子メールでお客様に配信されます。すぐに届かない場合は、忘れずにスパムフォルダをご確認ください。

## 組織用のPHPツールを購入するプロセスはどのようになっていますか。 

注文プロセスが簡単にできます：

2.	購入するライセンスの種類を選択します：単独ユーザー、チーム又は企業選択したライセンスの下にある「今すぐ購入」ボタンをクリックします。大規模な組織の場合は、お客様の要件に応じて対応いたしますのでお問い合わせください。
3.	これで、ショッピングカートに商品が表示されます。選択したものが正しい場合、下にスクロールし、請求情報と支払いオプションの入力を続けます。
4.	支払いオプション（デビット/クレジットカード、銀行振込など）を選択し、「続く」をクリックして注文を完了します。
5.	注文が行われ、支払いが承認されると、ライセンスキー番号が電子メールでお客様に配信されます。すぐに届かない場合は、忘れずにスパムフォルダをご確認ください。

## 御社のベンダーはどの支払い方法を受け入れますか。

多数な方法ができます。クレジット/デビットカード（Visa、Visa Electron、MasterCard、Maestro、Eurocard、Diners Club、American Expressなど）、Paypal、銀行振込、注文書などもっと見る： http://www.avangate.com/standard/vendor-faq.php#top14 

## 世界中からの支払いができますか。 

はい、支払いは、カードが発行された通貨に関係なく、システムで受け入れられているロゴのいずれかの下に登録されているクレジットカードでしたら、どの国からでも行うことができます。もっと見る： http://www.avangate.com/standard/vendor-faq.php#top14

## ウェブサイトの価格にはすでに税金が含まれていますか。

いいえ。VAT/ GST（物品・サービス税）レートは国によって異なるため、最終的なVAT / GSTの金額はショッピングカート内の価格にのみ追加されます。

## 購入注文を受け入れますか。

はい、購入注文は受け付けますが、商用ライセンスのみです。購入注文で支払いたい場合は、ソフトウェアを直接販売するのではなく、認定ベンダーのAvangate / 2CheckOutを通じて販売します。

## 購入注文を申請するにはどうすればよいでしょうか。 

簡単です。次のステップに従ってください。

1.	既にライセンスを購入している場合は、当社の店名を記入するか、更新リンクを使用してください。

2.	希望のライセンス数を選択してください。

3.	請求の詳細を記入してください。

4.	「お支払い方法」として「購入注文」を選択してください　。

 ![Payment Options](imgs/payment.png)

5.	通貨を選択します（購入注文にはUSDまたはEURのみ使用可能）

6.	独自の購買発注伝票を使用する場合は、内部購買発注番号を入力します。そうでない場合は、日付と署名のみを追加していただく購入注文の事前入力フォームが提供されます。

7.	製品のエンドユーザーになる対象からの配送先住所を入力します（または、請求の詳細と同じ場合何もして必要ない）

ご注文後、確認メールと製品の配送に関する詳細情報が届きます。

## 注文の処理にはどのくらい時間がかかりますか。

購入発注は最も遅い支払い方法です。通常、注文部門が受け取ってから処理に1〜2営業日かかります。

購入注文を処理するために修正が必要な場合は、お客様にご連絡することがあります。

## オンラインで購入した後、ライセンスを受け取りませんでした。ライセンスはどこにありますか。

スパムフォルダーを既に確認しており、ライセンスキーを通知するメールがそこにない場合は、次の手順に従い、何が起こったのかを確認していただくようお願いします。

1.	まず、支払い確認を確認し、注文の支払い取引が完了したことを確認します。
2.	支払いが正常に完了した場合は、注文で指定されたメールアドレスをご確認ください。それが正しい場合は、ご注文に矛盾がある可能性がありますので、ご注文のステータスを確認するためにご連絡いただくようお願いします。メールアドレスが間違っていると思われる場合は、ご注文の詳細をご修正ください。
3.	オンライン支払いが失敗したことがわかった場合は、別の注文をするか、又は別の支払いオプションの詳細についてお問い合わせください。

## ライセンスを毎年自動的に更新する方法はありますか。

はい、ご要望に応じて定期請求モジュールを有効にできます。さらに、ライセンスの有効期限が近づいたとき、および支払いが承認されたときに、常に通知を受け取ることができます。

## 再販業者からライセンスを購入できますか。

はい、再販業者からPHPツールを購入できます。再販業者を介して行われた注文は、価格、支払い、納品条件など、その再販業者が適用する購入条件に該当することに注意してください。

## 払い戻しはどうですか。 

ご購入日から30日後まで返金いたします。

払い戻しを要求するには、注文に使用したメールアドレス、注文参照番号、請求書番号、その他の有用な購入関連情報をお送りください。

## 割引はありますか。

はい！商用ライセンスには多くのディスカウントがあります。[購入ページ](https://www.devsense.com/purchase)をご確認ください。
