# パーソナルライセンス

## パーソナルライセンスとは何ですか？

**パーソナルライセンス**は、個人開発者が自己資金で購入するオプションです。  
このライセンスは譲渡不可であり、企業や会社からの払い戻しを想定している個人には提供されません。  
パーソナルライセンスは購入者本人のみが使用できます。  
異なるデバイスにインストールすることは可能ですが、同時に複数のデバイスで使用することはできず、他の人が使用することもできません。  
また、パーソナルライセンスは再販業者には提供されません。

## パーソナルライセンスを購入すると何が得られますか？

パーソナルライセンスでは、以下の3つのオプションから選択できます：

1.) PHP Tools for Visual Studio  
2.) PHP Tools for VS Code  
3.) PHP Tools for All Platforms (VS + VS Code)  

すべてのオプションには以下が含まれます：

- **12か月のサブスクリプション**（購入したプラットフォームに対して、新しいバージョンとすべてのアップデートにアクセス可能）  
- **12か月のプロフェッショナルサポート***  
- **永久フォールバックライセンス**（購入時またはサブスクリプション期限の12か月前に利用可能なバージョンを無期限に使用可能）  

**新機能:** 月額サブスクリプションも選択可能になりました。  

- 月額払いで、アクティブな期間中は常にアップデートとサポートを受けられます。  
- **12か月連続で支払いを行うと**、永久フォールバックライセンスが自動的に付与されます。  
- 12か月未満で支払いが中断された場合、永久フォールバックは付与されません。  

サブスクリプション期間（年間・月額）が終了し更新しない場合、アップデートやサポートは受けられませんが、永久フォールバックライセンスでカバーされたバージョンは引き続き使用できます。

*サポートは、Visual Studio 2010-2022またはVS Codeで動作するDevsenseツールに対するものであり、Visual StudioのIDE自体のサポートではありません。

## パーソナルライセンスは商用目的や商業開発に使用できますか？

はい。パーソナルライセンスは、あらゆる種類の開発に使用できます。  
ただし、企業向けには提供されず、譲渡もできません。  
詳細は[ライセンス比較表](https://www.devsense.com/purchase)をご覧ください。

## 企業がパーソナルライセンスを使用できますか？

いいえ。パーソナルライセンスは、購入者本人のみが使用できるものであり、  
他の人や企業が使用することはできません。

## 自宅や職場など、異なるデバイスでパーソナルライセンスを使用できますか？

はい。異なるデバイスにインストールし、職場や自宅、ノートパソコンやデスクトップPCなどで使用できます。  
ただし、同時に複数のデバイスで使用したり、他の人が使用したりすることはできません。  
ライセンスを持つ本人が使用する場合に限り、異なるデバイスで利用可能です。
