# Proyectos de Código abierto

## Las licencias de PHP Tools son gratis para proyectos de código abierto 

Con PHP Tools y [Visual Studio Community 2022](https://visualstudio.microsoft.com/downloads/), los programadores obtienen un ambiente de desarrollo poderoso para PHP ¡sin costo alguno! 

Nuestras licencias para proyectos de código abierto le permiten utilizar PHP Tools for Visual Studio y PHP Tools for VS Code. 

### Requisitos para elegibilidad 

El proyecto debe: 

 * cumplir con la definición de [código abierto](https://opensource.org/docs/osd)
 * ser un proyecto activo desde al menos 3 meses 
 * publicar versiones y/o actualizaciones regularmente 

El proyecto no debe recibir ningún tipo de financiamiento por parte de organizaciones comerciales o ningún otro tipo de patrocinio pago. 

Los desarrolladores que quieran obtener su Licencia para proyectos de código abierto deben: 

 * Ser desarrolladores del equipo central del proyecto 
 * Ser contribuyentes regulares del proyecto 
 * No recibir compensación monetaria por trabajar en el proyecto 

Los desarrolladores no deben proveer ningún tipo de consulta o servicio de entrenamiento pago, ni distribuir versiones pagadas del software.
