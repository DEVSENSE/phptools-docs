# ライセンス概要

> **注意:** 2025年3月3日以降、PHP Tools for Visual Studio と Visual Studio Code は、以下に説明する同じライセンスモデルを採用します。以前のバージョンには影響はなく、それぞれのリリース時に適用されていたライセンスモデルが維持されます。

# 永続フォールバックライセンスモデル

PHP Tools for Visual Studio および Visual Studio Code では、**永続フォールバックライセンスモデル**を採用しています。これは以下を意味します：

- **年間サブスクリプション**を購入すると、購入時点で利用可能なバージョンの永続フォールバックライセンスが即時に付与されます。
- サブスクリプションの有効期間中は、すべての更新とサポートを利用できます。
- **更新を継続しない**場合、サブスクリプション失効の時点から **過去12か月に公開されたバージョン** を使用できます（永続フォールバックバージョン）。ただし、新しいバージョンの更新やサポートは受けられません。

- **月額サブスクリプション**の場合、12か月連続で支払った後、サブスクリプション開始時に利用可能だったバージョンの永続フォールバックライセンスが付与されます。サブスクリプションが失効した後も、このフォールバックバージョンを使用できますが、更新やサポートは利用できません。

## ライセンスタイプ

**商用ライセンス** は、法人（企業、営利・非営利団体、政府機関など）のためのオプションです。企業または組織内の任意の開発者が、購入したライセンス数を超えない範囲で任意のマシンにインストールして使用できます。詳細は [法人向け商用ライセンス](./commercial-license) をご覧ください。

**個人ライセンス** は、自身の資金で購入する個人開発者向けのオプションです。譲渡不可です。複数のコンピュータにインストール可能ですが、同時に使用することはできません。企業による購入や、企業からライセンス費用の払い戻しを期待する個人には提供されません。詳細は [個人向けライセンス](./personal-license) をご覧ください。

いいえ。PHP Tools のすべてのバージョンは当社のウェブサイトからダウンロード可能で、[ソフトウェアをアクティベート](https://docs.devsense.com/vs/installation/activation) するにはライセンスの購入が必要です。

当社の購入ページでのオンライン注文は通常すぐに処理されます。ライセンスキーは通常 2 時間以内にメールで届きます。スパムフォルダを確認し、2 時間以上経過しても届かない場合は [info@devsense.com](mailto:info@devsense.com) までお問い合わせください。
