# Estudiantes y profesores

## Las licencias de PHP Tools son gratis para los estudiantes y profesores

Con PHP Tools y [Visual Studio Community 2022](https://visualstudio.microsoft.com/downloads/), los estudiantes obtienen un ambiente de desarrollo poderoso para PHP ¡sin costo alguno! 

Nuestras licencias académicas le permiten utilizar PHP Tools for Visual Studio y PHP Tools for VS Code. 

### Requisitos para elegibilidad 

Cualquier estudiante que desee adquirir una licencia de PHP Tools for Visual Studio debe:
 
 * estar inscrito en un curso de estudios para licenciatura o diploma en una universidad, instituto o escuela secundaria reconocida; y también,
 * ser menor de 26 años de edad.

Para más información, contáctenos a [info@devsense.com](mailto:info@devsense.com) 

# Preguntas frecuentes 

## ¿Las licencias para estudiantes y profesores son 100% gratis? 

Sí, ofrecemos licencias académicas y licencias para aulas de clases tanto para estudiantes como para profesores, respectivamente, 100% gratis. Al aceptar su solicitud, los estudiantes y los miembros de la facultad recibirán acceso completo a nuestros productos por un año. Estas licencias son únicamente para propósitos estrictamente no comerciales. Por favor, lea nuestro [Contrato de licencia](https://www.devsense.com/es/purchase/licenses/classroom). 

## ¿Qué es una licencia académica? 

Una licencia académica es la opción para los estudiantes que utilizarán PHP Tools para propósitos educativos o para investigaciones académicas. Este tipo de licencia no puede ser utilizada para ningún tipo de desarrollo comercial. Por favor, asegúrese de leer nuestro [Contrato de licencia](https://www.devsense.com/es/purchase/licenses/academic). 

## ¿Qué es una licencia para aula de clase? 

Una licencia para aula de clase es la opción para los miembros del profesorado que utilizarán PHP Tools para propósitos educativos dentro de un aula de clase. Estas licencias no pueden ser utilizadas para ningún tipo de propósito comercial. Para obtener una licencia para aula de clase, los miembros del profesorado deben estar de acuerdo con nuestro [Contrato de licencia](https://www.devsense.com/es/purchase/licenses/classroom) y dar cierta información acerca de la institución. 

## ¿Cuál es la diferencia entre una **licencia académica** y una **licencia para aula** de clase?  

La licencia académica es una licencia individual para estudiantes únicamente para aprender. Las licencias para aula de clase se entregan a los miembros del profesorado para enseñar dentro del aula de clases. Ninguna de estas dos licencias pueden ser utilizadas para fines comerciales. 

## ¿Cuál es la diferencia entre una licencia **comercial** y una licencia para **aula de clase**? 

Nuestras licencias académicas y licencias para aulas de clase no difieren de las licencias comerciales en el sentido técnico. Con la licencia académica o para aula de clase tendrá acceso completo a todas las características y funciones del producto. La única restricción es el propósito de la licencia. Las licencias académicas y las licencias para aulas de clase pueden ser utilizadas únicamente con propósitos educativos, mientas que las licencias comerciales pueden ser utilizadas con propósitos comerciales. 

Para obtener una licencia para aula de clase, los miembros del profesorado deben estar de acuerdo con nuestro [Contrato de licencia](https://www.devsense.com/es/purchase/licenses/classroom) y dar cierta información acerca de la institución. 

## ¿Cómo puedo comenzar a utilizar sus productos? 

Por favor, envíenos un correo electrónico a [info@devsense.com](mailto:info@devsense.com) con una prueba de su estatus en la institución académica. Luego, una vez aprobada su solicitud, recibirá una licencia gratis sin costo alguno. Por favor, asegúrese de leer nuestro [Contrato de licencia](https://www.devsense.com/es/purchase/licenses/academic). 

## ¿Qué versiones del producto obtendré? 

Obtendrá todas las versiones nuevas de PHP Tools for Visual Studio y PHP Tools for VS Code durante el período de validez de su licencia, que es un año. 

## ¿La licencia académica es perpetua? 

No, la licencia académica es válida únicamente por un año. No se proporcionan licencias académicas perpetuas. Luego de este período, necesitará obtener una nueva clave de activación para el próximo año, o adquirir una licencia personal si ya no es estudiante. 

## Como estudiante, ¿puedo renovar mi licencia académica por otro año? 

Si aún es estudiante, podrá renovar su licencia por otro año. Lo puede hacer directamente desde su cuenta en Devsense. 

## ¿Puedo utilizar mi licencia en las computadoras de mi salón de clases? 

Si desea utilizar PHP Tools en un salón de clases, le ofrecemos la licencia para aula de clase. Por favor, incentive a su profesor a utilizar nuestros productos gratis en los salones de clases, para propósitos educativos. 

## Somos una institución educativa. ¿Son las licencias académicas la opción correcta para nosotros? 

Las licencias académicas son únicamente para los estudiantes. Las licencias para aulas de clase les permiten a los miembros del profesorado utilizar nuestros productos gratis en su aula de clase, con propósitos educativos. Por favor, contáctenos enviando un correo electrónico a [info@devsense.com](mailto:info@devsense.com). 

## ¿Existe algún descuento para estudiantes después de su graduación? 

Sí, los estudiantes que terminen sus estudios o que ya posean una licencia académica de PHP Tools son elegibles para obtener 25% de descuento en la compra de cualquiera de nuestros productos. 

## Soy un estudiante, pero también trabajo medio tiempo ¿Puedo utilizar mi licencia académica gratis para mis proyectos de trabajo?

No, nuestras licencias académicas son exclusivamente para propósitos no comerciales. Tiene la opción de adquirir una licencia personal o puede hablar con su empleador para que adquiera una licencia personal para los proyectos relacionados con su trabajo. 

## No soy estudiante, tampoco profesor, pero trabajo en una institución educativa y quisiera adquirir PHP Tools para desarrollar aplicaciones internas de la institución. ¿Son las licencias académicas la opción correcta para mí? 

No, nuestras licencias académicas son exclusivamente para propósitos educativos y no comerciales. Para universidades, institutos y secundarias reconocidas, ofrecemos 25% de descuento en nuestros productos para el desarrollo de sus aplicaciones internas. Contáctenos a [info@devsense.com](mailto:info@devsense.com) para más información.
